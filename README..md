# ❤️ Memories Forever – React Memory Journal

Memories Forever is a beautiful React web application where users can store, manage, and relive their special memories.

Users can register, log in, add memories with images, rate them, and view them in a memory dashboard.

The project is built using React.js, JavaScript, HTML, and CSS, with Local Storage used for saving user data.



## 📖 Introduction

Memories Forever works like a personal digital memory journal.

Users can:
- Create an account
- Add memories with images
- Rate experiences
- View memories in a dashboard
- Toggle between Light Mode and Dark Mode

It helps users keep all their special moments in one place.



## ✨ Features

### 🔐 User Authentication
- Register new account
- Login system using Local Storage

### 🏠 Home Page
- Welcome screen with project branding

### ➕ Add Memory
- Upload image or paste image URL
- Add title, location, date
- Write experience notes
- Add star rating

### 📊 Memory Dashboard
- Total memories counter
- Favorite memories counter
- Average rating calculation

### 🖼 Memory Cards
- Displays all saved memories
- Shows image, title, rating, location

### 🔍 Search Memories
- Quickly search saved memories

### 🌙 Dark / Light Mode
- Theme toggle option

### 📍 Location Map
- Map view for memory location

### 🔒 Protected Routes
- Only logged-in users can access memory pages
